# 材料机器学习自动诊断工具

面向材料/化学/能源科研数据的回归建模初筛工具。上传实验数据的 Excel/CSV，自动完成建模、图表、解释和诊断报告。

---

## 📦 下载与安装

### 第一步：安装 Python

> **Windows 用户**
> 1. 访问 https://www.python.org/downloads/
> 2. 下载 Python 3.9–3.12 版本（推荐 3.11）
> 3. 安装时 **务必勾选** ☑️ "Add Python to PATH"

> **Mac 用户**
> 1. 打开终端（Terminal）
> 2. 安装 Homebrew（如已安装可跳过）：
> ```
> /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
> ```
> 3. 安装 Python：
> ```
> brew install python@3.11
> ```

安装完成后，打开终端/命令提示符输入 `python --version`，显示版本号即表示成功。

---

### 第二步：获取本工具

**方式一：下载压缩包（推荐新手）**
1. 点击本页面右上角的「Code」→「Download ZIP」
2. 解压到任意文件夹（路径中不要有中文）

**方式二：Git 克隆**
```
git clone https://xxx/xxx.git
```

---

### 第三步：启动

#### 🪟 Windows 用户

**方式 A：双击启动（推荐）**
1. 进入解压后的文件夹
2. 双击 **`start.bat`**
3. 首次启动会自动安装依赖（约 2–5 分钟，取决于网速），**只需等待一次**
4. 完成后自动打开浏览器

**方式 B：命令行启动**
```
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
.venv\Scripts\streamlit run app.py
```

#### 🍎 Mac 用户

**终端启动：**
```bash
# 进入工具所在目录
cd /下载/解压后的文件夹路径

# 首次启动（后续只需最后一行）
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/streamlit run app.py
```

或在终端执行：
```bash
chmod +x start.sh
./start.sh
```

---

### 第四步：开始使用

启动后浏览器会自动打开 http://localhost:8501，页面长这样：

```
📊 Step 1: 数据加载与探索   ← 上传数据、查看质量
🤖 Step 2: 模型训练与评估   ← 训练模型、看结果
🔍 Step 3: 可解释性分析     ← SHAP 解释、偏依赖图
```

**推荐先点「加载示例数据」试试效果**，再上传自己的数据。

## 🖥️ 界面说明

### Step 1 — 数据加载与探索

| 模块 | 说明 |
|------|------|
| 上传数据 | 支持 CSV / XLSX / XLS，点示例可体验 |
| 数据概览 | 左边原始数据，右边统计量 |
| 数据质量检查 | 缺失值、异常值、近重复样本检测 |
| 选择目标列 | 选你要预测的那一列 |
| 数据预处理 | 类别编码、缺失删除 |
| 特征工程 | 特征筛选、归一化（默认 Z-Score） |

### Step 2 — 模型训练与评估

| 模块 | 说明 |
|------|------|
| 评估配置 | 测试集比例、交叉验证方式 |
| 超参优化 | Bayesian / Random / Hyperband / Grid（默认 Bayesian） |
| 训练 | 点击「开始训练」按钮 |
| 模型对比 | 查看 6 个模型的表现排名 |
| 可视化 | 预测散点图、HPO 收敛曲线 |
| 自动诊断 | A/B/C/D 等级判定 + 改进建议 |
| 下载报告 | TXT / HTML / PDF 三种格式 |

### Step 3 — 可解释性分析

- SHAP 摘要图 + 特征重要性（解释每个特征的影响）
- PDP 偏依赖图 + ICE 曲线（看特征如何影响预测）

## 📊 数据格式要求

| 要求 | 说明 |
|------|------|
| 文件格式 | CSV、XLSX、XLS |
| 表格结构 | 行 = 样本，列 = 特征 |
| 目标列 | 必须为**数值型**（如强度、导电率、硬度等） |
| 最少样本 | 建议 ≥ 30 行 |
| 类别特征 | 自动编码（类别数 ≤ 10） |

**示例数据格式：**

| cement | water | age | compress_strength |
|--------|-------|-----|-------------------|
| 540.0 | 162.0 | 28 | 79.99 |
| 380.0 | 165.0 | 90 | 61.89 |

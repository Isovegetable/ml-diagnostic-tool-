# 🧪 材料机器学习自动诊断工具

> 材料/化学/能源科研数据的 **机器学习建模工具** — 上传 Excel，自动出模型、出图表、出报告！

📌 **完全本地运行**，数据不上传网络，适合未发表论文和企业配方数据。

---

## ✨ 它能干什么？

| 功能 | 说明 |
|------|------|
| 📊 **数据探索** | 上传 CSV/Excel，自动统计数据质量 |
| 🤖 **自动建模** | 6 种模型 + 自动超参优化，选出最好的 |
| 🔍 **可解释性** | SHAP 分析、PDP 偏依赖图 |
| 📋 **诊断报告** | A/B/C/D 等级评分 + HTML/PDF 报告导出 |
| 🔒 **隐私安全** | 本地运行，不断网也能用 |

---

## 🚀 5 分钟快速上手

### 小白用户 → [点这里看完整教程](使用说明.md)

跟着教程走，10~15 分钟就能用起来（大部分时间在等待下载）：

```
① 装 Python（一次）→ ② 下载本工具（一次）→ ③ 双击启动（每次）
```

### 极速启动（懂命令行）

```bash
# 1. 进入本工具目录
cd ml-diagnostic-tool

# 2. 创建虚拟环境（首次）
python -m venv .venv

# 3. 安装依赖（首次，约 2-5 分钟）
.venv/bin/pip install -r requirements.txt  # Mac
.venv\Scripts\pip install -r requirements.txt  # Windows

# 4. 启动（以后每次运行这行就行）
.venv/bin/streamlit run app.py  # Mac
.venv\Scripts\streamlit run app.py  # Windows
```

启动后浏览器自动打开 http://localhost:8501，**建议先点「加载示例数据」试试效果**。

## 📚 三步操作

| 步骤 | 做什么 | 结果 |
|------|--------|------|
| **Step 1** | 上传数据，选目标列 | 数据概览 + 质量检查 |
| **Step 2** | 点击「开始训练」 | 最佳模型 + 诊断等级 A~D |
| **Step 3** | 查看 SHAP 和 PDP 图 | 特征重要性分析 |

## 📊 支持的数据

- **格式**：CSV、XLSX、XLS
- **结构**：行 = 样本，列 = 特征
- **目标列**：必须是数值（如强度、导电率、硬度）
- **最少样本**：建议 ≥ 30 行

| cement | water | age | compress_strength |
|--------|-------|-----|-------------------|
| 540.0 | 162.0 | 28 | 79.99 |
| 380.0 | 165.0 | 90 | 61.89 |

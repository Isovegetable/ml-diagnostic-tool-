# Materials ML Diagnostic Tool

A lightweight machine learning diagnostic tool for materials science, chemistry, and energy research data.

Upload your experimental data (Excel/CSV), and the tool will automatically run regression modeling, generate charts, provide SHAP/PDP explanations, and produce a diagnostic report.

---

## Quick Start

### 1. Install Python

**Windows:** Download from https://www.python.org/downloads/ — make sure to check **"Add Python to PATH"** during installation.

**Mac:** 
```bash
brew install python@3.11
```

Verify: open terminal and run `python --version`

### 2. Download

```bash
git clone https://github.com/Isovegetable/ml-diagnostic-tool-.git
cd ml-diagnostic-tool-
```

Or download ZIP from the GitHub page and extract it.

### 3. Launch

**Windows:** Double-click **`start.bat`** (first run takes 2-5 minutes for dependency installation)

**Mac/Linux:**
```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/streamlit run app.py
```

Then open http://localhost:8501 in your browser.

## How to Use

The app has 3 steps:

### Step 1: Data Loading & Exploration

1. Upload your Excel/CSV file, or click one of the example buttons to try with sample data
2. View data preview and statistics
3. Check data quality (missing values, outliers, near-duplicates)
4. Select the target column to predict (e.g., strength, capacity, conductivity)
5. Configure feature engineering (normalization: Z-Score by default)

### Step 2: Model Training & Evaluation

1. Click **"Start Training"** button
2. Wait for training to complete (a few seconds to a minute)
3. View model comparison results
4. Check the **diagnosis level** (A/B/C/D) — tells you whether your data is suitable for ML
5. Download reports (TXT / HTML / PDF)

### Step 3: Interpretability

1. Select a model to analyze (defaults to the best one)
2. View SHAP summary plot and feature importance bar chart
3. View Partial Dependence Plots (PDP) and ICE curves

## Data Format Requirements

| Requirement | Description |
|-------------|-------------|
| File format | CSV, XLSX, XLS |
| Structure | rows = samples, columns = features |
| Target column | Must be **numeric** (continuous values only) |
| Minimum samples | ≥ 30 recommended |
| Categorical features | Auto-encoded (≤ 10 categories) |

## Features

- 6 regression models: Linear Regression, Random Forest, Gradient Boosting, LightGBM, XGBoost, CatBoost
- 4 HPO methods: Bayesian TPE, Random Search, Hyperband, Grid Search
- K-fold cross-validation (5/10-fold or LOOCV)
- Data quality checks: missing values, outliers, near-duplicates
- Feature engineering: normalization (Z-Score/MinMax/Robust), low-variance filtering, high-correlation filtering
- SHAP analysis (beeswarm summary + feature importance bar)
- PDP partial dependence plots + ICE curves
- Diagnosis level A/B/C/D with actionable recommendations
- Report export: TXT / HTML / PDF

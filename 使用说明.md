# 🧪 材料机器学习诊断工具 — 安装使用教程

> **如果你完全不懂编程，跟着下面每一步做就能用！**
> 总共约 5 分钟，大部分时间是在解压压缩包。

---

## 📋 你需要什么

- 一台能上网的电脑（Windows 或 Mac 都可以）
- 一个要分析的数据文件（Excel 或 CSV 格式）

# 🪟 Windows 用户看这里

---

## 第一步：安装 Python（编程环境）

> Python 就像游戏的"运行库"，不装它这个工具跑不起来。

### ① 下载 Python

1. 打开浏览器，访问 **[https://www.python.org/downloads/](https://www.python.org/downloads/)**
2. 页面会自动识别你的系统，你会看到一个黄色的 **Download Python 3.xx** 大按钮
3. **点击这个黄色按钮**，下载安装包

> 📸 你会看到：网页顶部有一个黄色大按钮，写着 "Download Python 3.12.x"

### ② 安装 Python

1. 双击刚才下载的文件（文件名是 `python-3.12.x-amd64.exe`）
2. **⚠️ 最重要的一步！** 安装窗口打开后，**看最下方**，有一个复选框写着：
   - **`☐ Add Python to PATH`**
   - **务必把它勾上！变成 `☑ Add Python to PATH`**
3. 点击上方的 **「Install Now」**（立即安装）
4. 等几分钟，出现绿色字 "Setup was successful" 就装好了

> ⚠️ **没勾 "Add Python to PATH" 的话，后面会启动不了，只能重装！**

### ③ 验证是否装好了

1. 按键盘上的 **`Win` 键**（左下角 Ctrl 和 Alt 之间的那个）
2. 输入 **`cmd`**，回车
3. 会弹出一个黑色窗口（叫"命令提示符"）
4. 在里面输入以下命令，然后按回车：
   ```
   python --version
   ```
5. 如果显示 `Python 3.12.x` 这样的字，✅ 说明装好了
6. 如果提示"不是内部或外部命令"，说明上一步没勾 PATH，**重新运行安装程序，这次务必勾上**

---

## 第二步：下载本工具

### ① 下载压缩包

1. **打开分享的下载链接**
2. 下载 **`材料机器学习诊断工具-v1.0.0.zip`**（约 350MB）
3. 下载完成后，找到这个 ZIP 文件（通常在"下载"文件夹里）

> ⚠️ 文件比较大，下载需要一点时间，但里面已经包含了所有需要的依赖包，下载一次就不用再联网了。

### ② 解压

1. 右键点击 ZIP 文件，选 **「全部提取」**（或"解压到..."）
2. 选择解压位置：
   - ✅ **推荐：放在桌面**（解压后桌面上会多一个文件夹）
   - ❌ **不要放在中文名字的文件夹里**（比如"新建文件夹"）
3. 解压后得到这样一个文件夹，里面有很多文件和子文件夹

> 📸 文件夹里应该有：`app.py`、`packages/` 文件夹、`start.bat` 等

---

## 第三步：一键启动（推荐方式）

### ① 双击启动脚本

1. 打开解压后的文件夹
2. 找到 **`start.bat`** 这个文件（图标是齿轮⚙️或者白色文件）
3. **双击它**

### ② 等待启动完成

> 第一次启动只需 **几秒**，不用再联网下载！

- 你会看到一个 **黑色窗口**，显示：
  ```
  [OK] Starting Streamlit server...
  Starting... (1/15)
  Starting... (2/15)
  ```
- **不要关掉它**，等几秒后浏览器会自动打开 🎉

### ③ 浏览器自动打开

- 启动完成后，你的 **默认浏览器会自动弹出**
- 你会看到一个网页界面，这就是工具了 🎉
- **不会出现多个空白窗口**，只会打开一个页面

> ⚠️ 如果浏览器打开后白屏：
> - 直接按 **F5 刷新** 即可
> - 或者手动在浏览器地址栏输入：**http://localhost:8501**

### 以后再用

之后每次用，**只要双击 `start.bat` 就行**，秒开。关掉黑色窗口即可退出工具。

---

## 备用方案：命令行启动（如果双击 start.bat 失败）

如果双击报错，可以试试手动启动：

1. 打开解压后的文件夹，在地址栏（文件夹顶部的白色输入框）里
2. 输入 **`cmd`**，然后按回车
3. 在弹出的黑色窗口里，**依次输入以下命令**（每输完一条按一次回车）：

```bash
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt --no-index --find-links=packages
.venv\Scripts\streamlit run app.py
```

每条命令执行时：
- 第一条：几秒钟就结束
- 第二条：**也很快**（从本地缓存安装，不用联网下载）
- 第三条：启动工具，浏览器自动打开

# 🍎 Mac 用户看这里

## 第一步：安装 Python

### ① 打开终端

1. 按键盘上的 **`Command + 空格键`**
2. 输入 **`终端`**（或 Terminal），回车
3. 会打开一个白底或黑底的窗口

### ② 一行命令安装 Python

把下面这串字**全部选中复制**，粘贴到终端里，按回车：

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

- 这个叫 Homebrew，是一个安装工具
- 中间会让你输密码（输的时候屏幕不显示字符是正常的），输完按回车
- 安装大概需要 3~5 分钟

装完后，继续在终端里输入：

```bash
brew install python@3.12
```

等它跑完（1~2 分钟）

### ③ 验证

在终端输入：

```bash
python3 --version
```

如果显示 `Python 3.12.x` 这样的字 ✅ 就装好了

---

## 第二步：下载本工具

### ① 下载压缩包

跟 Windows 用户一样：
1. **打开分享的下载链接**（同 Windows 用户）
2. 下载 **`材料机器学习诊断工具-v1.0.0.zip`**
3. 下载完成后，双击 ZIP 文件会自动解压

> 建议把解压后的文件夹拖到 **桌面** 或 **文稿** 文件夹里

### ② 打开终端进入文件夹

```bash
cd ~/Desktop/文件夹的名字
```

> 💡 小技巧：输入 `cd `（cd后面有空格），然后把文件夹从 Finder 拖进终端窗口，会自动填入路径，再按回车

### ③ 首次启动

在终端里运行（**全程只需要运行一次**）：

```bash
# 创建虚拟环境
python3 -m venv .venv

# 从本地缓存安装依赖（不用联网，几秒钟）
.venv/bin/pip install -r requirements.txt --no-index --find-links=packages

# 启动工具
.venv/bin/streamlit run app.py
```

浏览器会自动打开，看到网页界面就成功了 🎉

> 以后再用，只运行最后一行 `pip install` 和 `streamlit run` 那步不需要重复执行。

# 📖 怎么用这个工具

启动后你会看到网页上分三步操作，**建议先点「加载示例数据」试试手**：

### Step 1：上传数据

1. 点击 **「浏览文件」**，选择你的 Excel 或 CSV 文件
2. 页面会显示表格预览和统计信息
3. 在 **「选择预测目标」** 下拉框中，选你要预测的那一列
   - 👉 比如你要预测"强度"，就选 `strength` 列
4. 其他设置保持默认就行，不用改
5. **先点「加载示例数据」** 体验一下也可以

### Step 2：训练模型

1. 点击 **「开始训练」** 按钮
2. 等几秒到几十秒（取决于数据量大小）
3. 页面会自动显示哪个模型效果最好
4. 诊断等级 A/B/C/D 会告诉你数据适不适合做预测
5. 点击 **「生成 PDF 报告」** 按钮，等几秒就会自动出现下载按钮
6. 也可选 **「生成 HTML 报告」**，两者内容一致（含模型对比、图表、建议）

### Step 3：可解释性分析

1. 查看 SHAP 图表（哪个特征对预测影响最大）
2. 查看 PDP 图（每个特征怎么影响结果）

# ❓ 常见问题（先看这个）

### Q：双击 start.bat 后黑色窗口一闪就没了

> 可能是 Python 没装好。
> 解决方法：在文件夹地址栏输入 `cmd` 回车，然后在黑框里输入 `start.bat`，能看到具体错误提示。

### Q：启动时提示"Python 未检测到"或"不是内部或外部命令"

> 说明安装 Python 时 **没勾 "Add Python to PATH"**。
> 重新运行 Python 安装程序，看到 ☑ Add Python to PATH 勾上再装一次。

### Q：浏览器打开后是白屏或显示"无法访问此网站"

> 刷新一下页面（F5），或者手动输入 http://localhost:8501

### Q：训练时卡住不动了

> 如果等了超过 5 分钟还没反应：
> 1. 关掉黑色窗口
> 2. 重新双击 start.bat 再试一次

### Q：结果 R² 很低，诊断等级是 C 或 D

> 这**不代表工具坏了**。不是所有数据都适合做机器学习。
> 可能原因：样本太少、数据噪音大、目标变量没有明显规律。

### Q：我的数据会不会被上传到网上？

> **完全不会。** 这个工具在你自己的电脑上运行，断网也能用。
> 你的数据不会离开你的电脑，适合未发表的论文数据和企业配方数据。

### Q：支持哪些文件格式？

> ✅ Excel（.xlsx / .xls）
> ✅ CSV（.csv）
> 文件内容格式：**行 = 样本，列 = 特征**

材料机器学习自动诊断工具 — 下载和使用说明
============================================


一、安装 Python（如果还没装）
-----------------------------

【Windows 用户】

1. 打开浏览器访问 https://www.python.org/downloads/
2. 下载 Python 3.11 或 3.12
3. 双击下载的安装包
4. 重要：安装时务必勾选「Add Python to PATH」
5. 一直点「Next」直到安装完成

验证是否装好：
- 按 Win + R
- 输入 cmd 回车
- 在黑框里输入 python --version
- 如果显示 Python 3.11.x 之类的版本号，就说明装好了

【Mac 用户】

1. 打开「终端」(Terminal)
2. 安装 Homebrew（如果还没装）：
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
3. 安装 Python：
   brew install python@3.11
4. 验证：终端输入 python3 --version，显示版本号即可


二、下载本工具
-------------

【方式一：下载 ZIP 压缩包（推荐）】

1. 打开工具所在的网页
2. 点击绿色的「Code」按钮
3. 点击「Download ZIP」
4. 下载完成后解压到电脑上任意位置
   - 建议放在桌面或文档文件夹
   - 路径中不要有中文
   - 好习惯：解压后把文件夹改名为 ml-tool

【方式二：用 Git 下载】

git clone https://github.com/xxx/xxx.git


三、启动工具
-----------

【Windows 用户 - 方法一：双击启动（最简单，推荐）】

1. 打开解压后的文件夹
2. 找到 start.bat 这个文件
3. 双击它
4. 第一次启动会比较慢（2-5 分钟），因为要自动安装依赖包
   - 你会看到一个黑色窗口在滚动文字，不要关掉它
   - 这是在下载和安装需要的软件包
5. 安装完成后，浏览器会自动打开一个网页
6. 如果浏览器显示「无法访问此网站」，刷新一下页面即可

以后再用：以后每次只需要双击 start.bat 就行了，直接秒开。

【Windows 用户 - 方法二：命令行启动】

python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
.venv\Scripts\streamlit run app.py

【Mac 用户】

1. 打开「终端」(Terminal)
2. 输入 cd 后面加空格，然后把解压后的文件夹拖进来，回车
3. 执行以下命令：

首次启动（只需执行一次）：
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt

每次启动执行这一行就行：
.venv/bin/streamlit run app.py

4. 浏览器会自动打开，如果没有，手动访问 http://localhost:8501


四、使用说明
-----------

启动后电脑屏幕上会显示一个网页，分三步操作：

【Step 1：上传数据】

1. 点击「浏览文件」按钮，上传你的 Excel 或 CSV 文件
2. 系统会自动显示数据预览和统计信息
3. 在「选择预测目标」下拉框里，选你要预测的那一列
4. 系统会自动检查数据质量（缺失值、异常值、重复样本）
5. 在「特征工程」区可以设置归一化方式（默认 Z-Score，不用改也行）
6. 也可以点击「加载示例数据」按钮，先看看效果

【Step 2：训练模型】

1. 点击「开始训练」按钮
2. 等待训练完成（几秒到几十秒）
3. 系统会显示哪个模型最好，以及各项指标
4. 页面最上方会有诊断等级（A/B/C/D），告诉你适不适合做 ML
5. 可以下载诊断报告（TXT / HTML / PDF 三种格式）

【Step 3：可解释性分析】

1. 选择要分析的模型（默认选最好的）
2. 系统会自动计算 SHAP 值
3. 查看 SHAP 摘要图和特征重要性柱状图
4. 查看 PDP 偏依赖图


五、常见问题
-----------

【启动时提示「Python 未检测到」】
说明 Python 没装好，或者安装时没勾选「Add Python to PATH」
重新运行 Python 安装程序，务必勾选「Add Python to PATH」

【双击 start.bat 后闪退】
可能是 Python 没装好
在 start.bat 所在文件夹的地址栏输入 cmd 回车
然后在黑框里输入 start.bat，能看到具体的错误信息

【训练时卡住不动】
如果是第一次运行，HPO（超参优化）会慢一些
如果超过 10 分钟还没反应，关闭窗口重新启动再试

【结果 R² 很低是怎么回事】
不是所有数据都适合做机器学习
诊断等级如果是 C 或 D，说明数据质量或信号不足
可能是样本太少、特征太乱、或者目标变量本身没有明显规律

【我的数据安全吗】
本工具完全在本地运行，数据不上传任何服务器
即使断开网络也能正常使用
适合未发表的论文数据和企业配方数据
